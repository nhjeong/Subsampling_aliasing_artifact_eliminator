# Subsampling_aliasing_artifact_eliminator
Artificial Intelligence-based Magnetic Resonance Image Reconstruction method using Generative adversarial network


## Environment
* Python 3.6.6
* Pytorch 0.4.1
* Numpy 1.15.2
* h5py, matplotlib


## Reference
